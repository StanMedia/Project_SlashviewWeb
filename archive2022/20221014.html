在Windows直接使用指令查詢記憶體資訊

每次要用到都要回去網路查找，乾脆記錄在此備用。

利用wmic指令查找記憶體的完整資訊

不用安裝任何軟體，請在Windows CMD下輸入列指令：

wmic memorychip get BankLabel, Capacity, ConfiguredClockSpeed, DeviceLocator, Manufacturer, PartNumber, SerialNumber, Tag

就可以得到下列輸出結果，當然這個結果一定是因電腦而異的啦！

BankLabel  Capacity    ConfiguredClockSpeed  DeviceLocator   Manufacturer  PartNumber   SerialNumber  Tag
BANK 0     XXX         1600                  ChannelA-DIMM0  XXX           XXX          XXX           Physical Memory 0
BANK 1     XXX         1600                  ChannelA-DIMM1  XXX           XXX          XXX           Physical Memory 1
BANK 2     XXX         1600                  ChannelB-DIMM0  XXX           XXX          XXX           Physical Memory 2
BANK 3     XXX         1600                  ChannelB-DIMM1  XXX           XXX          XXX           Physical Memory 3

補充說明

  1. 如果喜歡看HTML格式，可以在尾端添加「/format:hform > spec.html」，就會在執行目錄處寫入指定的HTML檔案，但我覺得大可不必。
  2. 如果要知道屬於哪種DDR規格，可以把PartNumber丟去Google就可以看到很多資料了。
