Windows Defender防毒軟體發出Win32/Hive.ZY威脅警告

今天用電腦的時候發現Windows Defender防毒軟體（Windows安全性）瘋狂發出win32/hive.zy警告，上網一查才發現這應該算是微軟推送病毒定義檔（安全性情報版本）的錯誤，將會導致所有使用到Chromium based或Electron包皮的瀏覽器APP，全部都會被誤報成中毒（Behavior:Win32/Hive.ZY），相關截圖如下所示。

已封鎖嚴重威脅 win32/hive.zy

Windows Defender警告之相關截圖如下：

目前這個事件已經在微軟的論壇上炸鍋了（win32/hive.zy removal notification every time I run a program），全球均有人回報，據悉是更新到 1.373.1508.0 病毒定義檔所致，目前微軟緊急釋出離線版本（1.373.1537.0）供給安裝，如下所示請謹慎服用：

64bit downloads
https://go.microsoft.com/fwlink/?LinkID=121721&arch=x64

32bit downloads
https://go.microsoft.com/fwlink/?LinkID=121721&arch=x86

後記

將安全性情報版本更新至1.373.1555.0之後，誤判的情況就解除了。
