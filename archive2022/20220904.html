在Windows桌面按右鍵，直接新增舊版EXCEL檔案（.xls）

EXCEL 97-2003工作表在現今已經很少被提及了，但是因為工作上需要在許多新舊系統之間切換，因此.xls檔案還是常常會使用到，特別是使用SQL匯出時期更常用到。由於每次要取得.xls都要很蠢的進行一次新增.xlsx後，再進入EXCEL另存成.xls，有夠浪費時間，因此這篇文章就是要教你怎麼把按下右鍵新增舊版EXCEL檔案（.xls）的能力找回來。

按右鍵新增舊版EXCEL檔案（.xls）

Step 1. 用現存的EXCEL軟體，使用另存新檔創造一個空白的舊版EXCEL（.xls）檔案，並將其命名為「EXCELOLD.XLS」。

Step 2. 將這個「EXCELOLD.XLS」放到「C:\Program Files\Microsoft Office\root\vfs\Windows\SHELLNEW」目錄下。如果你是安裝32Bits版本的Office，或許應該考慮目錄為「C:\Program Files (x86)\Microsoft Office\root\vfs\Windows\SHELLNEW」。

Step 3. 服用下方註冊檔：

Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\.xls\ShellNew]
"FileName"="C:\\Program Files\\Microsoft Office\\Root\\VFS\\Windows\\ShellNew\\excelold.xls"

Step 4. 重開機後跑去桌面按下右鍵，就可以看到「新增舊版EXCEL檔案」的功能啦！

Windows Desktop RightClick Add OldExcelFiles .xls DirectAddNewFiles XlsxConvertToXls
