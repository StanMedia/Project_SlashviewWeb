有關SSD混合碟快取的設定

最近混合碟大興其道，SSD也降價很多，有一些NB都開始推出混合式的硬碟，但是ReadyBoost其實超過2G後，就毫無加速效率，是否有更好的方式來儲存呢？

用新的晶片組的話，可以考慮使用 Intel IRST 技術，另外也可以考慮使用 ExpressCache（https://www.condusiv.com/partners/oem/technologies/expresscache/），是另外一個簡單且不錯使用的方案。

以 ASUS UX32VD 為例，我的做法是：

  1. 4G for 休眠
  2. 2G for Internet TEMP.
  3. Etc. partition for EXPRESS CACHE

ExpressCache設定教學

  1. 先清空SSD所有的partition區
  2. cmd > diskpart
  3. list disk
  4. select disk 1（端看你的SSD是第幾個編號而定）
  5. clean
  6. create mbr
  7. create create partition primary size=4096（取決你的記憶體大小，要塞入休眠狀態用）
  8. set id=84 override（設定這個分割區是休眠用）
  9. 再割一個簡單的磁區給Browser TEMP，這邊是割2G
  10. 安裝Diskeeper ExpressCache
  11. 安裝後該程式會馬上抓到這顆SSD最後沒有被用到的空間，並自動切出分割區。（沒有任何介面）
  12. 利用cmd進入到ExpressCache程式所屬位置
  13. reboot
  14. 輸入eccmd -info，查詢現在所有的快取狀況，有數值的話，就是開始在進行cache動作了
