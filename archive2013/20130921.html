VMware datastorage之*.vmdk更改檔案名稱

事實上在vSphere Client是沒有辦法幫vmdk檔案更改檔案名稱的，有可能是我用免費版本的關係吧！要幫vmdk改名字的話可以執行putty，利用SSH連線以console mode的方式進去ESXi 裡面改就可以了。

更改的方式如下：

Rename a virtual disk using a command similar to:

vmkfstools -E OldFileName.vmdk NewFileName.vmdk

改完會連動連隱藏關聯的檔案（-flat- like）名稱一起改，還蠻好用的，非常貼心！但是為何不做成UI介面呢？是打console text比較有屌感嗎？
