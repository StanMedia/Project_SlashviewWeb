Visual Studio 2012 MVC 4 project template lose?

MVC本來就是一個入門曲線很高的技術，當你按照書中一步步做卻發現，天啊，你根本沒有這個步驟的畫面時，你一定會很不爽。所以這一篇就是要寫Visual Studio 2012 Update 3 MVC 4 專案範本檔消失時，你要怎麼辦？

在這邊先抱怨一下，事實上，寫書的作者根本就是用他已經測過千百回的環境在快照圖片給讀者看，我只能說他的方法畫面只有他的電腦會出現，因為MVC的初學者，是不會以前有動到「ASO.NET實體資料模型」或者是「SQL Server Compact 4.0本機資料庫」的啦！也就是說，初學者的Visual Studio根本不會有這些捷徑，那你快照那些圖片中的捷徑是要叫人家點哪裡？拜託人家花錢買你的書，你要寫之前也先有道德一下重灌你的Visual Studio來模擬成初學者的環境好嗎？

好了，言歸正傳，一直試到死，Visual Studio 2012 更新到死，但就是沒有下列這張圖片的朋友，您有福了：

答案很簡單，就是到微軟的官方網站中下載擴充包，File Name:AspNetMVC4Setup.exe，File Size:35.9MB。

別介意他是for VS 2010的，總之下載裝完後把Visual Studio 2012關掉重新開啟就可以了。
