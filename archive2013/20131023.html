免費的作業系統

這件事情起源於蘋果電腦在2013/10/23宣佈Mac OS從此免費後，科技報橘這篇文章就出現這樣好笑的說辭與標題：「微軟更加繃緊神經！蘋果宣告「免費作業系統」時代來臨」。我只能說聽到免費就讚揚，就像看到黑影就開槍，還蠻無言的。

  1. 我能不能自由下載，並且移植到各家VM中來使用？可以的話這叫免費。

  2. 如果綁硬體的話也可以，我買的甲Mac電腦壞掉，能不能單買乙Mac純硬體方案，退我一些錢，然後我自己把OS裝到乙Mac裡面？如果可以，這也是某一種定義上的免費。

如果不是上面的做法，都不能叫做免費。

如果一台機器只能綁一套OS，我價格早就綁進去只是你看不到而已吧！然後你提供這個OS免費升等，那蘋果在2007年早就可以宣稱說iOS完全免費了啊... 就像微軟說我ServicePack免費，大家會有感嗎？還是大家覺得作業系統的更新檔修正本來就要收費比較合理？

一定會有人說，蘋果是指完全升級，也就是Windows 7跳到Windows 8完全免費，那我要反問一下Mac就不會發生iPhone舊機升不上去的窘境嗎？軟體跟硬體本來就是競爭關係，最終一定會遇到瓶頸的啊，為了使用者的最佳體驗，蘋果阻止升級這種做法我是贊同的。那如果最後蘋果把iOS上這套說法再搬過來MacOS這邊，這些人啟不被打臉乎？

我相信還是會有人繼續講，還是給升啊，升上去很慢的人就會換一台新的了。對！你有沒有發現你中了蘋果的陷阱了？所以蘋果給升級但是變慢，是自己電腦的問題，但微軟給升級但是變慢，是微軟程式寫的爛？

思路運行到這邊已經很明顯了，但最後這些人會講，事實上微軟沒有給過升級。對，這是事實，Windows 7換Windows 8是要錢的。但是如果我是微軟的話，我也會立即宣布我作業系統日後全面免費，但是要買我微軟牌的電腦才可以。（自己跳下來做PC硬體或者隨便跟OEM簽個約收掛牌費都可以）

然後我們再把時光拉到Windows XP時代：

  1. Windows XP Service Pack 1 重新命名為 Windows 7，使用者：YA！免費升級耶。
  2. Windows XP Service Pack 2 重新命名為 Windows 8，使用者：YA！免費升級耶。
  3. Windows XP Service Pack 3 重新命名為 Windows 9，使用者：YA！免費升級耶。
  4. Windows Vista 重新命名為 Windows 10，使用者：厚！跑起來超頓超爛的，一定是我的電腦太舊，反正都換了3個版本了，來買新電腦吧！

於是使用者去買了一台新的微軟牌電腦，內建Windows 10，且OS免費更新，請問使用者這樣被玩一輪後，到底得到什麼？

我的結論是行銷話語弄的好，生意變的很好做，使用者變得很好騙，你覺得呢？我們唯一能作的是等蘋果日後怎麼施行他承諾的事吧。搞不好蘋果連升了10年的作業系統，並讓10年前的機器依然可以跑的很順，那我這篇文章就自動變酸文嘍。

如果有一種免費的事物，必須建構在「平均上」更昂貴的消費上，那這對消費者來說是免費嗎？

20150122 Update

微軟今日宣佈Windows 7, Windows 8, Windows 8.1可以免費升級到Windows 10，而且不用綁硬體（任何PC都可以安裝Windows），不知道這些腦殘媒體要怎麼回應？未看先猜「蘋果壓的微軟喘不過氣，Windows 10終於宣告免費！」、「Windows XP用戶遭到Windows 10拋棄，用戶控訴微軟死要錢」之類的。
