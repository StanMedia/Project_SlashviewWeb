Windows Server 2008 R2磁碟清理工具安裝

手上還有幾台老舊的Windows Server 2008 R2正常的運作著，日前發現有一台的硬碟剩下不到2GB的容量，想必應該都被更新後的備份檔案佔光了，當下想要找個磁碟清理工具來大掃除一下時，發現又找不到了。

Windows Server磁碟清理工具安裝

正規的方式應該去安裝桌面體驗才對，但是裝這個又太肥大了，不如直接從資源檔案裏面硬幹吧！這裡採用的是Windows Server 2008 R2 64Bit作為範例，其他的版本八九不離十，請自己找一下應該就會找到對應的檔案目錄。

Step 1. 複製cleanmgr.exe到\System32\

copy C:\Windows\winsxs\amd64_microsoft-windows-cleanmgr_31bf3856ad364e35_6.1.7600.16385_none_c9392808773cd7da\cleanmgr.exe C:\Windows\System32\

Step 2. 複製cleanmgr.exe.mui到\System32\zh-TW\

copy C:\Windows\winsxs\x86_microsoft-windows-cleanmgr.resources_31bf3856ad364e35_6.1.7600.16385_zh-tw_c12f6dd4de8d11af\cleanmgr.exe.mui C:\Windows\System32\zh-TW\

Step 3. Windows+R，打入「cleanmgr」，熟悉的老朋友磁碟清理工具回來啦！

CleanManager cleanmgr.exe
