禁止Ubuntu下的Google Chrome瀏覽器自動更新

實在很受不了Google Chrome瀏覽器的摧殘，就算躲到了Linux Ubuntu下還是得要被強迫更新，超級惱人。去網路查詢了一下禁止Google Chrome自動更新的資訊記錄在此，不確定能不能完全禁止，日後若有結論再回來更新資訊。

禁止Linux Ubuntu Google Chrome瀏覽器自動更新

  1. 禁止軟體管理包更新
sudo apt-mark hold google-chrome-stable
  1. 透過策略禁止自動更新
sudo -i
touch /etc/default/google-chrome
nano /etc/default/google-chrome

然後在檔案裡面新增內容：

repo_add_once="false"
  1. 關閉Chrome元件的自動更新
sudo -i
mkdir -p /etc/opt/chrome/policies/managed
touch /etc/opt/chrome/policies/managed/component_update.json
nano /etc/opt/chrome/policies/managed/component_update.json

然後在檔案裡面新增內容：

{
  "ComponentUpdatesEnabled": "false"
}

理論上弄完後重新開機，應該就可以解決了（由於我已經更新到最新版所以無法證明），但我想Google Chrome應該不會就這樣輕易地放過我們的，等待日後有新的資訊再來補充。
