變更Ubuntu的IP位址

沒有任何營養，純粹記錄一下如何在Ubuntu上變更IP位址（IP Address）。

變更Ubuntu IP Address

Step 1. 先sudo -i取得最高管理權限。

Step 2. 切換目錄到/etc/netplan。

cd /etc/netplan

Step 3. 透過nano之類的文字編輯器，編輯名稱類似00-installer-config.yaml這種檔案，打開後會看到類似以下的內容：

network:
  version: 2
  ethernets:
    ens32 :
      dhcp4: false
      addresses:
      - 192.168.1.2/24
      routes:
      - to: default
        via: 192.168.1.1
      nameservers :
        addresses: [8.8.8.8, 1.1.1.1]

改成這樣格式的優點就是很好理解每一個參數的意義，ens32是網路介面的名稱，dhcp4是指定是否使用DHCP，addresses是指定IP位址，routes是指定路由，nameservers是指定DNS伺服器。缺點就是我竟然連要改一個IP位址都不會，只好去網路找資料。

Step 4. 儲存後重新開機應該就可以套用到新的IP位址。

感想

最後純粹發牢騷，Linux不同的發行版本就有不同的指令操作（我的知識還停留在ifconfig時代），在不想要安裝套件的狀態下唯一解就是去網路找新的指令，實在有夠討厭的。
