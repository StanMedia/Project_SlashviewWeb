在Javascript中使用類似C#的Region語法來折疊管理程式碼區塊

今日由於需要撰寫大量的Javascript，因此上網搜尋是否有類似C#的Region語法來折疊程式碼區塊，發現有一個不錯的解決方案，這個解決方案是使用註解來實現的。

在Javascript中使用Region語法摺疊程式碼

可以摺疊的JS語法如下，當然，這種客製化的東西肯定是編輯器自定義的，也就是如果程式語言編輯器不支援那就沒轍了，幸好經過測試VisualStudio、VisualStudio Code兩者我常用的編輯器都支援。

//#region 程式碼區塊註解

function test() {
  console.log("Hello World");
}

//#endregion

只要你在屬於Javascript的程式碼中加入//#region和//#endregion，就可以在編輯器中摺疊這段程式碼了。
