Linux系統下程式常用的載入等候動畫（Loading）

使用Linux下的相關程式，當需要進行載入等候的階段，常常會使用盲人點字符號繪製動畫（早期是使用/、|、\三個符號旋轉），想說蠻有趣的所以在C#環境下也實作一個來試看看。

透過C#實作Linux慣用的載入動畫

盲人點字符號事實上稱為布萊葉點字符號，下列的C# Console程式碼貼上去就可以看到動畫了，按下鍵盤的任意按鈕就可以結束程式。

Console.CursorVisible = false;
Console.OutputEncoding = System.Text.Encoding.UTF8;
//監控鍵盤
var bShouldExit = false;
_ = System.Threading.Tasks.Task.Run(() => bShouldExit = Console.ReadKey(true).KeyChar != '\0');
//布萊葉點字符號陣列
var cSpinnerChars = "⠋⠙⠹⠸⠼⠴⠦⠧⠇⠏";
//迴圈
for (var i = 0; !bShouldExit; i = (i + 1) % cSpinnerChars.Length)
{
  Console.Write($"\r{cSpinnerChars[i]} Loading ...");
  System.Threading.Thread.Sleep(100);
}
