在Ubuntu上面安裝Guest Additions

紀錄一下在Ubuntu（Linux）下安裝Guest Additions的過程。

在Ubuntu下安裝Guest Additions的步驟

Step 1. 你的HostOS一定要先安裝Guest Additions (VirtualBox Extension Pack)。

Step 2. 進入GuestOS後，先進行sudo -i切換到最高權限。

Step 3. 更新apt：

apt update
apt upgrade -y

Step 4. Guest Additions需要自行編譯核心模組：

apt install build-essential dkms linux-headers-$(uname -r) -y

Step 5. 掛載Guest Additions ISO 映像檔：請到GuestOS＞裝置＞插入Guest Additions CD 映像。

Step 6. 在GuestOS中輸入下列指令：

lsblk

應該會出現下列類似的訊息，這代表VBoxGuestAdditions.iso已插入但是尚未掛載：

sr0      11:0    1  57.4M  0 rom

將光碟掛載上去：

mount /dev/sr0 /cdrom

Step 7. 執行安裝Guest Additions

cd /cdrom
./VBoxLinuxAdditions.run

Step 8. 安裝後，請重新啟動GuestOS即可完成安裝：

reboot

Guest Additions安裝後的功能

  • 共享剪貼簿

這個功能應該是所有人最想要的功能了，但對不起，這個功能在VirtualBox原生檢視介面下，之於Ubuntu命令純文字介面（例如：Ubuntu Server）是沒有用的，他必須要在GUI介面下才可以使用。

  • 共享資料夾

可以在Ubuntu命令純文字介面使用，只要去VirtualBox的設定裡面設定好共享資料夾：

  1. 資料夾路徑：HostOS的資料夾路徑，請自行指定。
  2. 資料夾名稱：輸入shared。
  3. 掛載點：輸入/shared。
  4. 勾選自動掛載與設為永久。
  5. 到GuestOS中輸入下列指令，應該就可以看到HostOS分享的檔案了：
sudo -i
cd /shared
ls
