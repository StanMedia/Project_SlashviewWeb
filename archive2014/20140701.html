支援Windows XP, Server 2003的FileZilla FTP

全世界最知名的FileZilla開放源始碼FTP專案，自從2014年開始不再支援Windows XP以及Windows Server 2003了，只支援Windows Vista以後的作業系統，這讓很多仍然在使用Windows XP及Windows Server 2003的使用者來說，的確是一個很大的困擾。如果是說FileZilla Server版長時間開啟的話，有缺陷的問題就算了，但我使用XP只是想用Client端下載個小檔案就關掉了，有必要這麼的「不支援」嗎？我其實是不信這軟體底層用的Windows API有多先進就是了。

但是問題還是有解的，答案就是降版使用舊的版本。我特別跑去FileZilla官網找到暫存在SourceForge.NET的檔案，並調閱歷史取出舊有的最後支援Windows XP及Windows Server 2003的版本，並將它們陳列在下方，請安心下載服用。

FileZilla Server 0.9.42

FileZilla Client 3.8.0

如果你要長時間使用這個軟體，那你要小心有可能某些版本號有因為缺陷問題，讓你的FTP服務有漏洞。如果是臨時用的話我完全不會覺得有什麼樣的大問題就是了！我想我大概會持續使用FileZilla FTP Server很長很長的時間吧，畢竟有很多跑舊系統的機器不是你說換就可以換的（或者是根本沒有那個必要更換）！咳...逝者如斯夫，不舍晝夜...
