關閉Windows Server的IE增強式安全設定

若非正式用途，在一般的測試時刻，我們不太會需要Windows Server來幫我們進行IE上網的把關動作，因為有時候因為測試的需要，會去網路抓一些程式或檔案來進行設定與觀察。

在Windows Server 2008後，IE增強式安全設定換地方了，所以在此以Windows Server 2012進行記錄。
