Windows Server在登入畫面時取消Ctrl+Alt+Del之程序

Windows Server在登入畫面的時候，總會要求管理者要先來一下Ctrl+Alt+Del的經典三按鈕之呼叫，才會進入到詢問帳號密碼的登入模式，有一些人覺得這樣的步驟有點多此一舉，在這篇文章中　將介紹如何將其移除（取消）。

下面就是取消Ctrl+Alt+Del才能進入登入畫面之方法：

Setp 1. 「Windows + R」＞gpedit.msc，進入本機群組原則編輯器。

Setp 2. 進入電腦設定＞Windows設定＞安全性設定＞本機原則＞安全性選項＞互動式登入:不要求按CTRL+ALT+DEL鍵。

Setp 3. 雙擊點開，改成已啟用，並且按下確定。

Step 4. 登出後，就可以看到登入畫面乾乾淨淨啦！不會有任何要求使用CTRL+ALT+DEL鍵的字眼，要登入時只要在畫面的任何一個地方，按下滑鼠按鈕即可，如果是觸控螢幕的話，那當然是手指往上滑一下就好嘍！
