Windows Server改回傳統式登入介面（需輸入帳號密碼）

在Windows Server 2008之後，或許是介面友善設計使然，亦或許是支援觸控之方便性，總之，登入畫面上會大剌剌的列舉現在本機可登入的所有使用者，這對資訊安全來說是一項很大的諷刺，這無疑是告訴要惡搞你Server的人，你的本機到底有多少個帳號，以及帳號完整名稱（一字不漏）。

想當然還是回歸到像Windows Server 2003那種傳統的輸入帳號及密碼的介面，會來的比較安全一些，下面就是換回來安全式介面的方法：

Setp 1. 「Windows + R」＞gpedit.msc，進入本機群組原則編輯器。

Setp 2. 進入電腦設定＞Windows設定＞安全性設定＞本機原則＞安全性選項＞互動式登入:不要顯示上次登入的使用者名稱。

Setp 3. 雙擊點開，改成已啟用，並且按下確定。

Step 4. 登出後，就可以看到回歸到傳統式的登入畫面啦！（需要自己打入使用者帳號介面）此外，Title會被改成「Other User」。
