以Chromium為基底之瀏覽器（Edge、Chrome）快速鍵整理

因為有需要特別整理這些資訊，供給日後查找，也分享給需要的人。常用的快速鍵例如F11、F12之類的我就不列舉了，這邊只列舉我需要的部分。理論上來說所有的Chromium內核瀏覽器都適用才對，我使用的是Microsoft Edge，當然也不排除有其他發行商自己的實作就是了。

Edge、Chrome瀏覽器快速鍵

  • Ctrl + L：移動焦點到網址列

  • Ctrl + T：開啟新TAB

  • Ctrl + W：關閉當前TAB

  • Ctrl + TAB：循環右移TAB焦點

  • Ctrl + Shift + TAB：循環左移TAB焦點

  • Ctrl + N：開啟新Instance（開啟一個新的瀏覽器視窗）

  • Ctrl + Shift + N：開啟新隱私模式Instance（開啟一個新的瀏覽器視窗且為隱私模式）
