將劇迷（Gimy）網站中對兒童不適當的選單隱藏

最近小孩有使用劇迷 Gimy這個影音網站進行線上的卡通影片觀賞，但我發現上面有一些對兒童不適當的連結（成人、老司機、18禁...），因此透過油猴（Tempermonkey）寫了一個Script腳本，如果有這方面需求的家長，可以使用這個腳本來針對這個網站不恰當的內容進行隱藏。

安裝油猴瀏覽器插件（Tempermonkey plugin）

油猴（Tempermonkey）是一種瀏覽器套件，適用於各大瀏覽器使用，您可以到Chrome或Edge的擴充管理員中找到線上商店，以Microsoft Edge來說套件的中文名稱叫做竄改猴，但一般來說坊間還是統一用油猴來稱呼，請找到這個套件並安裝它。

匯入腳本至套件

要匯入腳本到油猴使其可以執行的方式有下列幾種，如果沒有特別的需求建議採用Ａ方式即可，日後該網站有異動更新而我恰好還在關注這個事情的話，我會直接過去更新腳本程式碼，你的瀏覽器打開後會自動偵測是否有新版，若有的話它會自動更新到新的程式碼腳本。

Ａ、線上匯入腳本（日後可線上自動更新）

直接打開油猴的管理介面，點選控制台（Dashboard）＞匯入匯出工具＞自URL安裝，將下列網址貼上後點選安裝即可。

https://content.slashview.com/file/20240820/GimyForKids.user.js

Ｂ、手動下載腳本：

下載下方GimyForKids.user.js腳本後，點選控制台（Dashboard）＞匯入匯出工具＞匯入＞選擇檔案，指定該檔案腳本後即可完成安裝。

隱藏Gimy劇迷網站兒童不宜連結

Ｃ、手動建立腳本：

如果擔心有甚麼閃失的話，可以複製下方程式碼自行建立全新腳本亦可，只是麻煩了點。

// ==UserScript==
// @name         Gimy for kids
// @namespace    http://tampermonkey.net/
// @version      2024-08-20
// @description  Prevent unnecessary content from appearing in children's view.
// @author       Slashview
// @match        *://gimy.ai/*
// @icon         https://www.google.com/s2/favicons?sz=64&domain=slashview.com
// @grant        none
// @run-at       document-end
// ==/UserScript==

(function() {
  'use strict';
  var oLists = document.querySelectorAll('ul.nav.navbar-nav > li');
  for (var i = 0; i < oLists.length; i++) {
    if (i > 5) {
      var oItem = oLists[i];
      oItem.parentNode.removeChild(oItem);
    }
  }
})();
