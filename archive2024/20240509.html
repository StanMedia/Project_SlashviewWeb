Ubuntu console下的圖形化檔案總管

很厭倦在console下不斷地輸入指令切換目錄，也很難透過單一指令綜覽現在整體檔案系統的情況（以及現在身處的目錄），突然想起早期在DOS時期就已經有類似圖形化（其實是用ASCII描繪的框架）的檔案總管、檔案管理員，為何在Linux世界沒有人想要透過簡單的方向鍵與Enter來尋訪檔案呢？

檔案管理員：Midnight Commander

不是沒有，只是我不知道，透過一番網路搜尋找到一套名稱很中二的Midnight Commander，執行檔案簡寫mc也很有記憶點。

安裝指令：

sudo apt-get install mc

完成後輸入下列指令就可以運行：

mc

薑薑～回到以前DOS時代的復古感，下面也有一排Function鍵讓你使用，例如按下F4就可以立刻檢視檔案。
