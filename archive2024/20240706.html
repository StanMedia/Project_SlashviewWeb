設定Microsoft Edge的快取位置與大小

Microsoft Edge的快取位置如果沒有特別設定，預設會被放在「C:\Users\User\AppData\Local\Microsoft\Edge\User Data\Default\」目錄下，分別以Cache、Code Cache這兩個資料夾存放。截至現在為止，在Edge的設定介面仍然沒有提供任何簡便的設定方式提供使用者轉移。

為何需要設定Microsoft Edge的快取位置

現在的電腦主要的運作磁碟大部分都是SSD，然而SSD最怕的是大量頻繁的寫入，這將會造成儲存單元的壽命快速流失，而日常進行的網頁瀏覽就是一種需要大量讀寫的行為，這時候我們就會希望把瀏覽器的快取位置搬出去作業系統主磁碟。你可以選擇打造一個RAM DISK或是安裝另外一顆SSD應對，延長主磁碟的壽命降低整顆掛掉的風險，畢竟主磁碟一崩潰整個作業系統與資料就毀了。

設定Microsoft Edge的快取位置（DiskCacheDir）與快取大小（DiskCacheSize）

可以透過regedit.msc註冊表來完成，可將下列的文字複製起來存成catch.reg執行即可。其中的快取大小單位為bytes，我將其設定為2GB（HEX:80000000； DEC:2147483648）：

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Edge]
"DiskCacheDir"="E:\\Cache\\MicrosoftEdge"
"DiskCacheSize"=dword:80000000

設定後可以的話先把所有的Edge瀏覽器都關閉，然後去一開頭描述的預設目錄，把Cache、Code Cache這兩個資料夾存放的內容都砍掉，因為日後不會再用到了。刪除原有快取後可以重開機，然後再去執行Edge，就可以看到對應的目錄（例如：E:\Cache\MicrosoftEdge）裡面開始出現暫存的快取資料了。
