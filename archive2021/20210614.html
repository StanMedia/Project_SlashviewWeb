jQuery document ready的各種寫法

以前也沒有很在乎，反正寫習慣了就好，後來發現身邊的人越來越愛用簡式寫，那就把它紀錄在這邊，以後就從善如流嘍。

第一型：我愛用的型態，一目了然沒甚麼好講的

$(document).ready(function() {
  //...
});

第二型：簡式，好像熟悉jQuery的人後期大都這樣寫了

$(function(){
  //...
});

$符號與jQuery符號可以互換的寫法就不贅述了。

第三型：引入ES6的[Arrow Functions](https://caniuse.com/?search=arrow functions)寫法，除了該死的IE之外好像都支援。

$(() => {
  //...
});

熟悉C# Lambda或expression對Arrow Functions應該都不陌生，目前先這樣，有新的寫法再回來更新。
