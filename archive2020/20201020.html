Chrome瀏覽器頂端列消失問題的解法

Chrome瀏覽器在某些時刻頂端列會消失，這個問題其實困擾我大約有兩年多了，消失的方式不是這個應用程式的頂端列消失，而是當Chrome應用程式開起來視窗放到最大（Window Maximum）的時候，上方Windows經典的三個按鈕、Chrome網址列、Chrome我的最愛列都會消失變成全黑，彷彿這個應用程式上方被某種黑色區塊遮罩一樣，非常的詭異。

若說是Chrome瀏覽器的問題的話，把瀏覽器縮小視窗後是沒問題的，將其拉到另外一顆螢幕也沒問題。若說是Windows的問題的話，在有問題的這顆主螢幕打開其他的應用程式，也是沒問題的。會發生這樣的情況也不是說一定是Chrome應用程式才會這樣（也有過其他應用程式也會有雷同的情況），但只要有這種情況發生，99.9%都發生在Chrome開啟成為工作焦點（Focus）的情況下，或者是工作列有Chrome瀏覽器的情況之下。狀況示意圖如下：

一般發生這樣的情況，我通常都會勉強地把手頭上的工作做完後，把電腦重新開機（其實登出就可以了）就解決了，但是今天剛好手頭上有一些比較重的工作根本不可能重新開機，一怒之下花時間上網找一下答案，沒想到這個問題大家都遇到了，而且把問題指向Intel內顯晶片（卡），Intel大概也嗅到狀況不對，就把鍋甩給Windows。

來看看Intel怎麼說： Maximized window blank/empty space on top of screen

看到Intel寫上這段話，我都快笑死了...

Investigation has shown that this issue occurs not only on Intel graphics controllers but also with 3rd party graphics vendors.

This is a strong indicator of an issue with the Operating System.

好啦，我個人認為這個鍋不應該是Windows或Intel背負，因為我自己用的是Nvidia顯示卡也是中獎，我強烈懷疑是現代瀏覽器啟用GPU加速所致，而我們又不可能為了這個不常出現的狀況把硬體加速關閉，所以我傾向朝著Scale Full Screen來調查。

※ 備註：自從Microsoft new Edge改用Chromium核心後，這個問題也開始出現了（原本的original Edge不曾發生過）。

20211004 補充說明：

偶然在網路上看到有一組神秘的快速鍵（Windows + Ctrl + Shift + B）可以快速重啟Windows的圖形驅動程式，如果你是使用Windows 10的話，我認為或許這組快捷鍵可以取代上面的程序（但後來其實我不太遇到這個問題了，純粹推測這組快捷鍵有用，個人沒實測過），這組快捷鍵也確定可用在Windows 10的萬年「夜間光線」失效BUG上。

20220309 更新：最近又某然發生一次頂端列消失（黑化、空白）問題，確定服用「Windows + Ctrl + Shift + B」可以快速解決問題。
