SSMS專用的SQL Code Formatter程式碼格式化套件

最不願意做的事就是在Microsoft SQL Server Management Studio（SSMS）做SQL的編輯撰寫工作，真的是很搞不懂為何這邊的IDE編輯器萬年不進化？但是身為程式設計師沒辦法，你永遠躲不了在這邊需要進行程式碼編輯的命運，於是就是ApexSQL Refactor出場的時候啦！

首先請到下列網址ApexSQL Downloads，然後在畫面中找到ApexSQL Refactor，這個套件是在進行程式碼格式化用的，完全免費，請用力的下載吧！

下載後直接點兩下，基本上如果你不是Administrator是安裝不了的，因此請你在該程式圖像上點選右鍵選「以系統管理員身分執行」就可以了。安裝過程中MSSMS必須被關閉，安裝完成後請重新執行MSSMS，你就會發現選單上面有出現ApexSQL了，如下圖所示。

你可以用選單去進行Format SQL，也可以用快速鍵「Ctrl + Alt + Shift + F」，亦可以自建快速選單，在UI上面點一下按鈕就好了。

如果要細部格式的調整，可以到ApexSQL > Options中進行調整就好。

MSSMS SSMS MicrosoftSQLServerManagementStudio SQLCodeFormat SQLCodeFormatter
