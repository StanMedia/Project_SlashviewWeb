SKYPE文字格式之使用方式

如果你經常在SKYPE上面貼上程式碼與朋友、同事交談，一定會對於SKYPE將程式碼中的一些命中表情符號的代碼，轉換成表情圖示感到困擾，這篇文章就是在教你如何規避這種情況，以及使用其他的格式來強調語句。

SKYPE 粗體（bold）

一種文字格式的表現方式，在符號中的文字，都將成為粗體。

使用此符號將可表示：粗體

SKYPE 斜體（italic）

一種文字格式的表現方式，在符號中的文字，都將成為斜體。

使用此符號將可表示：斜體

SKYPE 刪除線（strikethrough）

一種文字格式的表現方式，在符號中的文字，都將被畫上刪除線。

使用此符號將可表示：~刪除線~

SKYPE 帶表情純文字（no text formatting）

在符號後的文字，將會被去除文字格式的表現方式，但仍會顯示表情符號（emotion icons）。

在此符號之後：@@ 粗體不會有效果，但是仍然可以使用(H)表情符號。

SKYPE 行內純文字（inline plain text）

在單行文字之中，出現在符號中的文字，都將成純文字。

在此符號之中，所有文字都將視為純文字，例如： １．這是一段{code}程式碼{code}分享給你。 ２．這是一段```程式碼```分享給你。

SKYPE 全文純文字（full plain text）

出現在符號後的文字，全部都將成純文字。

在此符號之後，所有文字都將視為純文字，例如：
這裡有一大段程式碼分享給你：
!!
static void Main(string[] args)
{
  var H = "null";
  Console.WriteLine(H);
  //BraBra....
}
